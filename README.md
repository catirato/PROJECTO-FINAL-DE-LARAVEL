**MusicBands**

A **MusicBands** é uma aplicação web desenvolvida em Laravel para gestão e consulta de bandas musicais e dos seus álbuns. O projeto permite organizar informação de forma simples e intuitiva, com uma área pública para visualização de bandas e discos, e uma área reservada para utilizadores autenticados e administradores.

Na aplicação, os visitantes podem explorar a lista de bandas, consultar os detalhes de cada uma e ver os álbuns associados. Os utilizadores autenticados têm acesso a funcionalidades adicionais, como a edição de álbuns, enquanto os administradores podem gerir completamente o conteúdo da plataforma, incluindo criar, editar e remover bandas e álbuns.

Este projeto foi desenvolvido com foco na prática de conceitos de **Laravel**, como rotas, controllers, models, autenticação, middleware, validação de formulários e gestão de ficheiros.
